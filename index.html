Calibration Analytics Web Application – Scrum Backlog with Estimated Man Hours

This file consolidates the full Scrum backlog for building a .NET web application that replaces the Excel-based calibration analytics workbook and adds QuickBooks integration. It covers the entire scope from project setup through deployment, including ingestion, parsing, mapping, normalization, manual regression, statistics, ICV verification, reporting, QA approval, testing, and accounting sync.

The backlog reproduces behavior currently spread across these workbook areas:

  • Cal Data – raw calibration run ingestion and parsing
  • ICV Data – verification run ingestion and validation
  • Ref Table – method configuration, thresholds, regression options, and level mappings
  • Report Writer – lookup-style derivations, internal standard summaries, normalization support
  • DVD – manual regression engine, weighting, point inclusion and exclusion, predicted values, residuals, RSE, r/r²
  • Summary Report – final acceptance rules, pass/fail status, and reporting layer
  • QuickBooks – customer sync, invoice-ready payloads, invoice creation, and status sync

Total Stories: 49

Story 9.3 – Compute manual weighted linear regression with intercept
16 hours
As an analyst, I want calibration regression to be calculated manually using defined weighted least squares formulas so that the system reproduces the Excel workbook’s regression outputs.
Manual RegressionWeighted Least SquaresDVD
Implementation Notes
  • Do not use third-party regression libraries for this core algorithm.
  • Compute and persist the intermediate sums.
  • Handle denominator-zero conditions as validation failure.
S0  = Σ w
Sx  = Σ w*x
Sy  = Σ w*y
Sxx = Σ w*x²
Sxy = Σ w*x*y

Den = (S0 * Sxx) - (Sx²)

Slope (m) = ((S0 * Sxy) - (Sx * Sy)) / Den
Intercept (b) = (Sy - (m * Sx)) / S0
Acceptance Criteria
  • The system computes and stores S0, Sx, Sy, Sxx, Sxy, Den, slope, and intercept.
  • If Den equals 0, the computation fails with a clear validation error.
  • The regression result matches a known reference dataset within a documented numeric tolerance.
Story 9.4 – Compute manual weighted linear regression forced through zero
10 hours
As an analyst, I want the regression model to optionally be forced through zero so that the application can comply with methods that require zero intercept bias.
Manual RegressionForced ZeroDVD
Implementation Notes
  • When ForceZeroIntercept=true, set intercept to zero and compute slope using the forced-zero weighted formula.
Sxx0 = Σ (w * x²)
Sxy0 = Σ (w * x * y)

Slope (m) = Sxy0 / Sxx0
Intercept (b) = 0
Acceptance Criteria
  • The stored intercept value is always zero in forced-zero mode.
  • If Sxx0 equals 0, the computation fails with a clear validation error.
  • The regression output explicitly indicates forced-zero mode.
